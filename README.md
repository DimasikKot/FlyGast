
## Как это работает?
Когда вы садитесь на счастливого гаста вы получаете статус "строитель".
Получив данный статус вы в определённом (конфигом) радиусе (по умолчанию 200) получаете полёт.

Если вы отойдёте слишком далеко, то полёт отключится и вы получите эффект левитации на 20 секунд, чтобы безопасно приземлиться.
Вы можете прописать команду `/fly` и вручную отключить статус строителя. (эффект левитации в таком случае не выдаётся)

## Конфиг

```yaml
# v1.2 - это версия конфига, обновите, если на сервере используется старая версия

radius: 200                         # Дальность работы полёта игрока от гаста
slow-down-on-left-the-radius: true  # Включать ли плавное падение на 20 секунд после принудительного отключения статуса
flights-into-normal: true           # Разрешён ли полёт в обычном мире?
flights-into-nether: true           # Разрешён ли полёт в аду?
flights-into-the-end: true          # Разрешён ли полёт в эндер мире?
flights-into-other: true            # Разрешён ли полёт в других мирах?

message:
  send-on-exit: true                              # Отправлять ли сообщение в конце езды на гасте?
  on-exit: "§aСтатус «строитель» активирован"     # Сообщение в конце езды на гасте
  send-on-command-fly: true                       # Отправлять ли сообщение после команды `/fly`?
  on-command-fly: "§cСтатус «строитель» отключён" # Сообщение после команды `/fly`
  send-on-left-the-radius: true                   # Отправлять ли сообщение после принудительного отключения статуса?
                                                  # Сообщение после принудительного отключения статуса
  on-left-the-radius: "§cСтатус «строитель» отключён. Вы улетели слишком далеко от гаста"
```
